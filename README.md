## How to use

In your terminal, type:

```bash
  git clone https://github.com/itamcampos/vimfiles.git ~/.vim
```

Then:

```bash
  echo "source ~/.vim/vimrc" >> ~/.vimrc
```

Run Vim and type

```bash
  :PlugInstall
```
